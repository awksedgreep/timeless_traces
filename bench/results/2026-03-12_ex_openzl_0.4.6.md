# TimelessTraces Benchmarks — ex_openzl v0.4.6
**Date:** 2026-03-12
**Platform:** 10-core Apple Silicon Mac

## Compression Sweet Spot (block size vs ratio)

| Spans/block | Raw Size | Compressed | Ratio | Per-Span |
|-------------|----------|------------|-------|----------|
| 10 | 6.1 KB | 1.2 KB | 5.0x | 124.6 bytes |
| 50 | 30.0 KB | 4.0 KB | 7.51x | 81.7 bytes |
| 100 | 60.3 KB | 7.5 KB | 8.07x | 76.5 bytes |
| 500 | 302.5 KB | 34.9 KB | 8.67x | 71.5 bytes |
| 1000 | 602.3 KB | 68.7 KB | 8.77x | 70.3 bytes |
| 5000 | 3.0 MB | 341.9 KB | 8.84x | 70.0 bytes |

Sweet spot: ~500-1000 spans/block (ratio plateaus, per-span cost minimized)

## Ingestion Pipeline (500K spans)

| Path | Throughput |
|------|------------|
| Writer only | 177.0K spans/sec |
| Writer + Index | 500.3K spans/sec |
| Full pipeline (Buffer -> Writer -> Index) | 360.5K spans/sec |

## Notes

- Writer+Index is faster than writer-only due to batch amortization in the
  combined path
- Full pipeline throughput of 360K spans/sec on a single 10-core node covers
  the vast majority of use cases
